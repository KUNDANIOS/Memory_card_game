# 🃏 Memory Card Game

A classic memory card matching game built with **React**. Flip cards, find matching pairs, and test your memory!

## ✨ Features

- Flip cards to reveal hidden symbols/images
- Match pairs to keep them revealed
- Track your moves and score
- Shuffle cards on every new game
- Responsive design for desktop and mobile

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- npm (comes with Node.js)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/KUNDANIOS/Memory_card_game.git

# 2. Navigate into the project directory
cd Memory_card_game

# 3. Install dependencies
npm install

# 4. Start the development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000) in your browser.

## 🕹️ How to Play

1. Click any card to flip it over.
2. Click a second card to try to find its match.
3. If both cards match, they stay face-up.
4. If they don't match, both cards flip back face-down.
5. Keep going until all pairs are matched!
6. Try to complete the game in as few moves as possible.
